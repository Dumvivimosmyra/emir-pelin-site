## 🚀 **SONRAKI ADIM: GITHUB DEPLOYMENT**

### **GitHub'a Yükleme Rehberi**
1. **Repository Oluştur**
   - GitHub.com'a git
   - "New repository" tıkla
   - İsim: `emir-pelin-site`
   - Public seç
   - README ekleme (zaten var)

2. **Dosyaları Yükle**
   ```bash
   git init
   git add .
   git commit -m "İlk commit - Romantik site tamamlandı"
   git branch -M main
   git remote add origin https://github.com/[kullanici]/emir-pelin-site.git
   git push -u origin main
   ```

3. **GitHub Pages Aktifleştir**
   - Repository → Settings
   - Pages bölümü
   - Source: Deploy from a branch
   - Branch: main seç
   - Save

4. **Site Canlı!**
   - URL: `https://[kullanici].github.io/emir-pelin-site`
   - 5-10 dakika içinde aktif olur
